## Installation

To get started with env-vault, you'll first need to make sure you have all the prerequisites installed on your system. Then, you can proceed to install env-vault using your preferred package manager. Below are the instructions for different installation methods:

### Using npm

If you're using npm as your package manager, you can install env-vault by running the following command in your terminal:

```
$ npm install -g env-vault
```

Using env-vault is simple and intuitive. First, you'll want to make sure it's installed. Then you can start using it. Here's a basic example:
